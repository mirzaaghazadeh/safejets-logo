# SafeJets Logo Animation

A single-file HTML/Canvas rendering of the SafeJets logo with two infinite-loop animations.

**Demo:** https://mirzaaghazadeh.github.io/safejets-logo/

## Animations

- **Animate** — bars cycle highlight via opacity (active = 100%, others = 50%).
- **Animate + Move** — same cycle, plus the highlighted bar slides ~30px to the right.

Click a button to start the loop, click again to stop and smoothly reset.

## Run

Just open `index.html` in any browser. No build, no dependencies.
